# Third-Party Notices

This project depends on the following third-party Python packages, installed via pip:

## Key Dependencies

- **requests** — HTTP library for making API calls
- **feedparser** — RSS/Atom feed parsing
- **python-dotenv** — Environment variable loading
- **anthropic** — Anthropic Claude API client
- **pyaml** — YAML serialization
- **pydantic** — Data validation using Python type annotations

All dependencies are installed at runtime via pip install -r requirements.txt and are not bundled in this repository.

See equirements.txt for the complete list of direct and transitive dependencies with pinned versions.
